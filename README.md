# WeatherNow

A clean weather dashboard built with **React**, **Vite**, **TS**, and **Tailwind CSS**.

### Features

- Search weather by city name
- Current temperature, feels like, humidity, and wind speed
- Dynamic weather emoji based on conditions
- Responsive dark-themed UI

### Usage

1. Get a free API key at [OpenWeather](https://openweathermap.org/api)
2. Create `.env` with `VITE_OWM_KEY=your_key`
3. Run:

```bash
npm install
npm run dev
```
